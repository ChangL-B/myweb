html을 배운 후에는?




html 수업을 모두 수강한 여러분 축하드립니다. :)

우리도 이제 웹페이지를 만들수 있게 되었어요.

매일 네이버, 구글과 같은 웹페이지를 보기만 하다가
나만의 웹페이지를 만드니까 기분이 어떠신가요?

어떤 분들은 나만의 홈페이지를 만드니까 뿌듯한 감정이 들수도 있고,
또, 어떤 분들은 여기서 더 나아가 자신의 홈페이지를 더 예쁘게 만들고 싶으실 꺼에요.

여기서 홈페이지를 더 예쁘게 꾸미고 싶으신 분들은 css라는 언어를 배우시면 도움이 되실꺼에요.

css는 html의 연장선이라고 보시면 되는데, 우리가 배웠던 html 태그에 css를 통해 속성을 추가할수 있어요.

예를 들어, 글씨의 색상, 홈페이지의 색상같은것을 css를 통해 바꿀수 있답니다.

혹시 홈페이지를 추가적으로 디자인 하고 싶으신가요? 그렇다면 생활코딩의 css 강의를 통해 쉽게 배우실수 있어요. 그러니까 우리 같이 가지 않을래요? :)
